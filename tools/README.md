# Tools

这里放可复用的数据处理工具模块。

建议后续按以下顺序补齐：

1. `dxf_parser.py`: DXF -> Raw Geometry JSON
2. `dxf_renderer.py`: DXF/JSON -> PNG
3. `geometry_normalizer.py`: Raw Geometry JSON -> Normalized Geometry JSON
4. `graph_builder.py`: Semantic Diagram JSON -> Topology Graph
5. `validator.py`: JSON schema 与字段合法性检查
6. `rule_engine.py`: 工程规则校验
7. `json_to_dxf.py`: Semantic JSON -> DXF
